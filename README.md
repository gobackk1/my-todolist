### `npm start`

- ローカル開発する時に使います
- [http://localhost:3000](http://localhost:3000)がオープンします
- 編集するとブラウザがリロードします
- console に出るエラーは確認してください

### `npm run test:e2e`,

- `npm start` してサーバーを立てた後に実行してください
- テストする時に使います
- プルリク、マージ前はテストを通してください

### `npm run test:unit`

- テストする時に使います
- プルリク、マージ前はテストを通してください

### `npm run build`

- プロダクション用にビルドします

### `npm run deploy`

- firebase でデプロイします
